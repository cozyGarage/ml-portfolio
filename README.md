# Machine Learning Portfolio

This repository is my umbrella portfolio for learning machine learning through project-based practice using **Scikit-Learn**, **PyTorch**, and related tools.

It is organized as a long-term learning journey rather than a single project. Each project focuses on one major skill area from the book *Hands-On Machine Learning with Scikit-Learn and PyTorch*.

## Goals
- Learn machine learning by building projects
- Practice end-to-end workflows: data loading, preprocessing, training, evaluation, and communication
- Build a GitHub portfolio that becomes stronger over time
- Develop a repeatable habit of reading, coding, experimenting, and documenting results

## How I Study in This Repo
1. Read one chapter or concept block from the book
2. Build or extend one small project
3. Write down what I learned in the corresponding project README
4. Save experiments in notebooks
5. Refactor stable code into `src/`

## Recommended Weekly Rhythm
- 3 days: reading + coding
- 1 day: debugging and experiments
- 1 day: writing notes and cleaning the repo

## First Steps
- Read `docs/roadmap.md`
- Read `docs/study-plan.md`
- Start with `projects/01-housing-price-predictor/README.md`
- Use `notebooks/` for experiments before moving code into `src/`
