# Study Workflow Guide

## Core Loop

For every project:
1. Read the concept
2. Explore in notebook
3. Move stable code to src/
4. Train (Colab if needed)
5. Save artifacts to Drive
6. Update README

## Before Each Session
```bash
git pull origin main
```

- read project README
- choose ONE small goal

## During Session
- focus only on that goal
- experiment and test

## After Session
```bash
git add .
git commit -m "progress"
git push origin main
```

- save large outputs to Drive
- update notes

## Important
Consistency > speed.
