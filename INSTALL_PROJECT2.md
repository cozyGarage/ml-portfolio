# Install Project 2 into Your Repo

From your repo root, copy the contents of this bundle into the repository root.

If you unzip this archive inside `ml-portfolio/`, the files will land in the correct paths.

Main entry point after install:

```bash
python scripts/run_project2_mnist.py --stage classical
```

Colab / PyTorch path:

```bash
python scripts/run_project2_mnist.py --stage pytorch --artifact-dir /content/drive/MyDrive/ml-portfolio-artifacts/project2
```
