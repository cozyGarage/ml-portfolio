# Project 3 Plan

## Goal
Build a repeatable model benchmarking workflow.

## Stage 1
- run the benchmark on the wine dataset
- inspect accuracy and macro F1
- identify the best model

## Stage 2
- repeat on breast cancer dataset
- repeat on iris dataset
- compare whether the best model changes by dataset

## Stage 3
- write down what makes a benchmark comparison fair
- summarize when different models win

## Deliverable
A lightweight benchmark tool plus notes on model selection.
