# EDA Checklist

## Data Understanding
- inspect shape
- inspect columns
- inspect target distribution
- review summary statistics
- check missing values

## Visual Checks
- histograms for numeric columns
- scatter plots for selected features vs target
- correlation matrix

## Questions to Answer
- which features seem most related to target?
- are there outliers?
- are scales very different across features?
- is target skewed?

## Output
Write a short summary after EDA:
- most important feature guess
- biggest data concern
- first modeling idea
