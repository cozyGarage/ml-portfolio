# Housing Project Plan

## Goal
Predict California housing prices with a clean baseline regression workflow.

## Phase 1 — Data Understanding
- load dataset from scikit-learn
- inspect shape and columns
- review summary statistics
- inspect correlations
- understand target variable

## Phase 2 — Baseline Modeling
- split train and test sets
- train Linear Regression
- train Decision Tree Regressor
- train Random Forest Regressor
- compare RMSE values

## Phase 3 — Documentation
- write down the best model
- explain why it performed better
- list next improvements

## Deliverables
- baseline training script
- one notebook for exploration
- a README update with results
